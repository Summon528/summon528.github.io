[UVA] 10341 - Solve It

2017-11-06

出處

https://uva.onlinejudge.org/index.php?option=onlinejudge&page=show_problem&problem=1282

題意

求 $p \times e^{-x} + q \times sin(x) + r \times cos(x) + s \times tan(x) + t \times x^2 + u = 0$ 在 $ 0 \leq x \leq 1 $的解
且輸入保證 $ 0 \leq p, r \leq 20 $、$ -20 \leq q, s, t \leq 0 $

思路

原方程式微分後得到
$ -pe^{-x} + q \cdot cos(x) + (-r \cdot sin(x)) + s \cdot sec^2(x) + 2tx $
根據題目的限制，導函數永遠$\leq 0$，原函數是遞減函數
題目還給了$x$的範圍，直接二分搜答案
