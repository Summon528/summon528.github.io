[UVA] 10306 - e-Coins

Cpp DP UVA
2017-12-11

目錄

  1. 出處
  2. 題意
  3. 思路
  4. 程式碼

出處

https://uva.onlinejudge.org/index.php?option=onlinejudge&page=show_problem&problem=1247

題意

給定m個硬幣，每種硬幣有兩個值conventional value和InfoTechnological value
求使用最少硬幣，且滿足 $S = \sqrt{C^2+I^2}$的硬幣數量
其中C代表選擇的所有硬幣的conventional value總和
I代表代表選擇的所有硬幣的value和InfoTechnological value總和
硬幣不限次數

思路

二維DP 無限背包
DP[J][K]代表要達成conventional value == J且InfoTechnological value == K的最少方法數
轉移dp[j][k] = min(dp[j - coins[i].first][k - coins[i].second] + 1,dp[j][k]);
i代表使用前i個硬幣
另外要判斷一下有沒有方法可以達成dp[j - coins[i].first][k - coins[i].second]可以才轉移
最後跑一圈For找出答案在哪裡
複雜度 $O(m \times S^2)$
